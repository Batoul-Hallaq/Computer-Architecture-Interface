# Software/Hardware Interface Project

This project explores the interaction between logical design and hardware performance. It includes designing logic circuits and conducting performance benchmarks on different computing architectures.

## Logic Design Tasks
* **Circuit Construction:** Designed and simulated complex logic circuits using **Logisim Evolution**.
* **Boolean Optimization:** Applied Boolean algebra to minimize circuits to their simplest forms.
* **Hardware Implementation:** Identified required TTL chips (e.g., 74x86, 74x04) for physical circuit construction.

## Hardware Benchmarking
A comparative analysis of different devices (Lenovo 83DV vs. MacBook Air) using the **Geekbench** suite:
* **Multi-Core Performance:** Evaluated processing power for programming tasks.
* **Clang Benchmark:** Measured CPU efficiency in code compilation.
* **Asset Compression:** Tested data handling and file reduction speeds.

## Files
* `SW-HW Project Report`: Detailed technical report of all tasks and results.
* `A.circ`: The initial logic gate simulation.
* `B.circ`: The Hardware Implementation Plan (Transistor-Transistor Logic)
